# Counter App

## Step 1 — Understand what you are building

### What does this app do?

A user clicks buttons to increase or decrease a number, and that number stays saved even after closing or refreshing the browser.

## What is the user flow?

- User opens the page → sees a number (starts at 0, or whatever was last saved)
- User clicks "+" → number goes up by 1
- User clicks "−" → number goes down by 1
- User clicks "Reset" → number goes back to 0
- User refreshes the page → the number is still the same as before refresh
